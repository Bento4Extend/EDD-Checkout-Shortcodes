This plugin will allow for you to extend your usage of Easy Digital Downloads through the use of shortcodes and to customize your display of the checkout.

== Description ==

This plugin will allow you to show, either on a download's page or off, different download ids.

The following shortcodes are available for your usage:

[download\_checkout\_count]
* This will return the number of items in the checkout box.

[download\_checkout\_total]
* This will return the total cost of all of the items in the checkout.

[download\_checkout\_loop] .. [/download\_checkout\_loop]
* This will create a loop that will loop through all items in the checkout box. It’ll allow the proceeding shortcodes to be used to show the info. To be able to do more you can look at the premium version called <a href="http://extend.bt4.me/downloads/edd-checkout-extendify/">EDD Checkout Extendify at: http://extend.bt4.me/downloads/edd-checkout-extendify/</a>

[download\_checkout\_item\_name]
* This will return the name for the current checkout item in the loop.

[download\_checkout\_item\_total]
* This will return the total price for the current checkout item in the loop.
